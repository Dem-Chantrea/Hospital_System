**🏥 Hospital Management System**

  This is a desktop-based Hospital Management System developed in C# as part of a university project. The system is designed to manage and organize various hospital   operations efficiently, providing a centralized solution for handling patients, staff, medical records, and administrative tasks.
The application is structured using multiple forms, each responsible for a specific module within the system, ensuring a clear and modular design.

--- 

📋 Overview <br>
  >💻 Built using C# and Windows Forms <br>
  🗄️ Database integration for data management<br>
  🧩 Modular form-based architecture <br>
  ⚙️ Covers both administrative and medical workflows <br>

  ---
  
🚀 Features<br>
  >👨‍⚕️ Doctor Management (FormDoctor)<br>
  >🧑‍🤝‍🧑 Patient Management (FormPatient)<br>
  >🧑‍💼 Staff Management (FormStaff)<br>
  🚚 Supplier Management (FormSupplier)<br>
  🦠 Disease Records (FormDisease)<br>
  💊 Medicine Stock Management (FormMedicineStock)<br>
  💳 Payment & Billing System (FormPayment)<br>
  📅 Appointment Scheduling (FormAppointment)<br>
  📝 Prescription Management (FormPrescription)<br>
  📦 Order Management (FormOrder)<br>
  🏥 Treatment Tracking (FormTreatment)<br>

  ---
  
🛠️ Technologies Used <br>
  >⚡ C# (.NET Framework)<br>
  🖥️ Windows Forms<br>
  🗃️ SQL Server / Local Database<br>

  ---
  
📌 Purpose <br>
    This project demonstrates the practical use of object-oriented programming, database connectivity, and GUI design in building a real-world hospital management system.
